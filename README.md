# 微机教室在线考试系统

面向微机教室局域网环境的在线考试系统。教师机在教室中作为服务器，学生机通过浏览器登录答题。

## 功能特性

- **题型支持**：选择题、判断题
- **学生端**：登录、在线答题（倒计时）、查看个人成绩与答题详情
- **教师端**：学生管理、考试管理（自定义题目）、发布/结束考试
- **成绩查询**：教师查看全班成绩，学生查看个人成绩
- **自动统计**：考试结束后自动批改并生成统计分析（平均分、及格率、分数段分布、各题正确率）
- **精美界面**：Vue 3 + Element Plus 现代化 UI

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Python FastAPI + SQLAlchemy |
| 数据库 | SQLite（无需额外安装） |
| 前端 | Vue 3 + Vite + Element Plus + Pinia |
| 认证 | JWT Token |

## Windows 打包（exe 安装版）

无需安装 Python / Node.js，教师机双击即可使用。

### 获取安装包

**方式一：自行打包（需 Windows 电脑）**

```bat
packaging\build_windows.bat
```

生成文件：
- `dist\微机教室考试系统\ExamSystem.exe` — 绿色版程序
- `release\微机教室考试系统.zip` — 可直接分发
- `release\微机教室考试系统_Setup.exe` — 安装版（需 Inno Setup，运行 `packaging\build_installer.bat`）

**方式二：GitHub Actions 自动构建**

在 GitHub 仓库 Actions 页面下载 `微机教室考试系统-Windows` 构建产物。

### 教师机使用

1. 解压 zip 或运行安装程序
2. 双击 **`ExamSystem.exe`** 或 **`启动考试系统.bat`**
3. 浏览器自动打开，学生机访问窗口显示的 IP 地址（如 `http://192.168.1.100:8000`）
4. 关闭命令行窗口即停止服务

数据保存在：`%APPDATA%\微机教室考试系统\`

详细说明见 [packaging/README.md](packaging/README.md)

## 快速开始（开发模式）

### 环境要求

- Python 3.10+
- Node.js 18+（仅构建前端时需要）

### 一键启动

```bash
# 1. 安装后端依赖
pip install -r backend/requirements.txt

# 2. 构建前端
cd frontend && npm install && npm run build && cd ..

# 3. 启动服务器（教师机运行）
python run.py
```

启动后终端会显示访问地址：
- 教师机本地：`http://localhost:8000`
- 学生机访问：`http://<教师机IP>:8000`

|------|--------|------|
| 教师 | teacher | teacher123 |
| 学生 | student01 ~ student05 | 123456 |

## 使用流程

### 教师操作

1. 使用教师账号登录
2. 在「学生管理」中添加/管理学生账号，或使用**批量导入**一次性录入多名学生
3. 在「考试管理」→「创建考试」中自定义题目（选择题/判断题），支持**批量导入题目**
4. 点击「发布」开放考试
5. 考试结束后点击「结束」，系统自动批改并统计
6. 查看「统计」和「成绩」了解考试情况

### 批量导入

**学生导入格式（CSV/TXT）：**
```csv
用户名,姓名,班级,密码
student06,张三,微机1班,123456
student07,李四,微机2班,
```
密码留空时使用默认密码（123456）。支持粘贴文本或上传 `.csv` / `.txt` 文件。

**题目导入格式一（CSV，适合 Excel 编辑）：**
```csv
类型,题目,选项A,选项B,选项C,选项D,答案,分值
choice,以下哪个是CPU,内存,CPU,硬盘,显卡,B,10
judge,RAM断电后数据不丢失,,,,,错误,10
```

**题目导入格式二（块格式，适合直接粘贴）：**
```
[选择] 以下哪个是CPU？
A. 内存
B. CPU
C. 硬盘
D. 显卡
答案：B
分值：10

[判断] RAM断电后数据不丢失。
答案：错误
分值：10
```

### 学生操作

1. 使用学生账号登录
2. 在「考试大厅」选择进行中的考试
3. 答题并提交（时间到自动提交）
4. 在「我的成绩」查看分数和答题详情
5. 在「打字练习」进行日常打字训练（不限次数）

### 打字考核标准

参考中考信息技术及义务教育信息科技课标：

| 等级 | 速度要求 | 说明 |
|------|----------|------|
| 达标 | ≥ 10 字/分钟 | 中考信息技术基本要求 |
| 良好 | ≥ 20 字/分钟 | 高于中考要求 |
| 优秀 | ≥ 30 字/分钟 | 课标第三学段基础达标 |
| 卓越 | ≥ 40 字/分钟 | 远超课标要求 |

准确率要求 ≥ 95%

## 开发模式

前后端分离开发：

```bash
# 终端 1：启动后端
cd backend && uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 终端 2：启动前端开发服务器
cd frontend && npm run dev
```

前端开发服务器会自动代理 API 请求到后端。

## 局域网部署说明

1. 确保教师机防火墙允许 8000 端口入站
2. 学生机浏览器访问 `http://教师机IP:8000`
3. 所有数据存储在教师机的 `backend/exam_system.db` 文件中
